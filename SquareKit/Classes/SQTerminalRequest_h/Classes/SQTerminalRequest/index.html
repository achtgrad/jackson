SQTerminalRequest

Encapsulates a URL request to the Square application.

Superclass: NSObject
Declared In: SQTerminalRequest.h

Discussion

An SQTerminalRequest instance will pre-fill a payment in the Square application. Use the properties to configure this request.


Methods

-send:

Initiates a Square payment

-URLRepresentation

send:

Initiates a Square payment 

- (BOOL)send:(NSError **)error;
Parameters
error

If an error occurs, will contain an NSError object that contains the error. Pass nil if you do not want error information.

Return Value

Returns YES if the request was successfully sent, NO otherwise.

Discussion

This method will build a request URL and attempt to open it in the Square application.

URLRepresentation

- (NSURL *)URLRepresentation;
Return Value

A URL that represents the current SQTerminalRequest instance.

Discussion

Builds a URL that conforms to the Square Terminal API specification from the current configuration of the SQTerminalRequest instance.
